# 📌 Bookmarklet Library

React 19 기반의 모던 북마크릿 라이브러리입니다. Emotion CSS-in-JS를 사용하여 모든 스타일을 JavaScript에 번들링합니다.

## 🚀 특징

- **React 19 기반**: 최신 React 기능 및 Hooks 활용
- **CSS-in-JS**: Emotion을 사용한 완전한 스타일 캡슐화
- **단일 파일**: 별도의 CSS 파일 없이 모든 것이 하나의 JS 파일에 포함됨
- **Drag & Drop**: 북마크바에 드래그하여 쉽게 설치
- **모듈형 아키텍처**: 단일책임원칙(SRP)에 따른 컴포넌트 설계
- **TypeScript**: 완전한 타입 안정성
